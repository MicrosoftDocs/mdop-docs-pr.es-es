# Cómo desinstalar componentes MED-V


En esta sección se explica cómo desinstalar el cliente y el servidor de MED-V.

## Cómo desinstalar el cliente MED-V


**Para desinstalar el cliente de MED-V**

1.  Si usa WindowsXP:

    -   En el panel de control, Abra **Agregar o quitar programas**.

2.  Si usa Windows Vista:

    -   En el panel de control, Abra **desinstalar un programa**.

3.  Seleccione **virtualización de escritorio de Microsoft Enterprise**y haga clic en **desinstalar**.

4.  Se recomienda eliminar la carpeta de la máquina virtual de MED-V. (La carpeta predeterminada es *C:\\MED-V imágenes*).

## Cómo desinstalar el servidor MED-V


**Para desinstalar el servidor MED-V**

1.  En el panel de control, Abra **Agregar o quitar programas**.

2.  Seleccione **Microsoft Enterprise Desktop Virtualization (servidor)** y haga clic en **desinstalar**.
